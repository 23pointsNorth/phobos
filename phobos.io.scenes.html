phobos.io.scenes.smurfs.exportSMURFScene(entities, path)¶

Exports an arranged scene into SMURFS. It will export only entities with a valid entity/name, and entity/type property.

Parameters:
  • selected_only (bool) – If True only selected entities get exported.
  • subfolder (bool) – If True the models are exported into separate subfolders
